📊 NCR Ride Bookings Analysis – Power BI
📌 Project Overview
This project analyzes ride booking data to understand revenue trends, booking performance, cancellation patterns, and customer ratings using Power BI.

📊 Key Insights
Total Bookings: 148K+
Total Revenue: 52M
Completion Rate: 62.51%
Cancellation Rate: 37%
Peak Revenue Time: Evening (5PM–9PM)

📈 Features
Interactive dashboard with filters
KPI indicators for business performance
Revenue and cancellation analysis
Customer rating insights

🛠 Tools Used
Power BI
